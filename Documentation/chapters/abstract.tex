The rapid urbanization and increasing mobility demands have led to growing interest in shared economy solutions, particularly in the vehicle rental sector. Traditional vehicle rental services often involve intermediaries, high costs, and limited accessibility, creating barriers for both vehicle owners and renters. RideShareX addresses these challenges by providing a peer-to-peer vehicle rental platform that connects vehicle owners directly with potential renters, eliminating the need for traditional rental agencies.

RideShareX is implemented as a full-stack web application using the MERN stack (MongoDB, Express.js, React, Node.js). The platform integrates Google OAuth 2.0 for authentication, JWT-based session management, a Multer-based image upload pipeline for vehicle photos, a booking lifecycle with conflict checks, and a demo payment system to simulate transactions. The frontend uses React and Tailwind CSS to provide a responsive, mobile-friendly user experience.

The project demonstrates the feasibility of a localized peer-to-peer vehicle rental platform that reduces costs for renters and increases earning opportunities for vehicle owners. Future enhancements include real-money payment gateway integration, vehicle ratings and reviews, GPS-based tracking, and native mobile applications to improve reach and user engagement.
